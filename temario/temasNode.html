Tema 2: Servidor HTTP

Con el módulo `http` de Node.js, puedes crear un servidor web básico sin necesidad de Apache o Nginx. 

const http = require('http');
const server = http.createServer((req, res) => {
  res.end('Hola Mundo desde Node!');
});
server.listen(3000);
