Tema 1: Introducción a CSS

Lección 1: ¿Qué es CSS?

CSS (Cascading Style Sheets): Lenguaje para diseñar y dar formato a HTML.

Define el aspecto visual (color, fuente, layout). Se vincula al HTML mediante la etiqueta <link> o <style>.

Tema 2: Diseño de Cajas

Lección 1: Modelo de Caja (Box Model)

Todo elemento en CSS es una caja que consiste en:
- Contenido: El texto o imagen real.
- Padding: Espacio interno.
- Border: Borde alrededor del padding.
- Margin: Espacio externo entre elementos.

Tema 3: Diseño Responsivo

Lección 1: Media Queries

Permiten aplicar estilos diferentes según el dispositivo (móvil, tablet, desktop).
Es la clave para que la web se vea bien en todas partes.
Ejemplo: @media (max-width: 600px) { ... }
