Tema 1: Introducción a CSS

leccion 1: ¿Qué es CSS y como se relaciona con HTML?

● CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para aplicar estilos y dar formato a documentos HTML.

● CSS se utiliza para definir el aspecto visual de los elementos HTML, como el color, el tamaño de fuente, el fondo y la disposición.

● Se relaciona con HTML al vincular archivos CSS externos a un documento HTML utilizando la etiqueta < link > o al aplicar estilos directamente en el documento HTML utilizando la etiqueta style

Tema 2: Diseño de cajas y posicionamiento

leccion 1: Modelo de caja CSS.


● El modelo de caja CSS describe cómo se renderiza y se muestra un elemento HTML.

● Consiste en el contenido, el relleno (padding), el borde (border) y el margen (margin) alrededor del contenido.

● Puedes ajustar el tamaño y el espacio de un elemento manipulando estas propiedades.

Tema 3: Media queries y diseño responsivo

leccion 1: Uso de media queries para adaptar la página a diferentes dispositivos.


● Las media queries son reglas en CSS que permiten aplicar estilos diferentes según las características del dispositivo en el que se visualiza la página, como el ancho de la pantalla.

● Se utilizan para crear diseños responsivos y adaptativos, optimizando la experiencia de usuario en diferentes dispositivos, como teléfonos móviles, tabletas y computadoras de escritorio.

Tema 3: Media queries y diseño responsivo

leccion 2: Diseño responsivo utilizando CSS

● El diseño responsivo es una técnica que permite que una página web se adapte y se vea bien en diferentes tamaños y resoluciones de pantalla.

● Se logra utilizando unidades de medida flexibles (porcentajes, ems) en lugar de unidades fijas (píxeles) y aplicando reglas CSS específicas para diferentes rangos de tamaño de pantalla.

● Los elementos y componentes de la página se redimensionan y reorganizan automáticamente para ajustarse al espacio disponible.
