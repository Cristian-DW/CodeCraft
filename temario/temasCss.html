1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de diseño.

Si HTML es el esqueleto, CSS es la piel, la ropa y el maquillaje. 

selector {
    propiedad: valor;
}

/* Ejemplo */
p {
    color: blue;
    font-size: 16px;
}
La palabra "Cascada" significa que el orden importa: si escribes dos reglas iguales, la última gana.

2. Colores

En el diseño digital, los colores son matemáticas:

  • Hexadecimal: `#FF5733` (Rojo, Verde, Azul en hex).
  • RGB: `rgb(255, 87, 51)` (Rojo, Verde, Azul en decimal).
  • HSL: `hsl(0, 100%, 50%)` (Hue, Saturation, Lightness). Es el más humano e intuitivo al diseñar.

3. Tipografía

El texto es el 90% de la web. 

body {
    font-family: 'Inter', sans-serif; /* Tipo de letra */
    font-size: 16px; /* Tamaño base */
    font-weight: 700; /* Grosor (Bold) */
    line-height: 1.5; /* Altura de línea (legibilidad) */
}
Usa fuentes Sans-Serif (sin remates) para pantallas, ya que son más fáciles de leer que las Serif (tipo periódico).

4. Modelo de Caja (Box Model)

Todo elemento HTML es una caja rectangular compuesta por capas (de adentro hacia afuera):

  1. Content: El texto o imagen.
  2. Padding: Relleno interno (aire entre contenido y borde).
  3. Border: El borde visible.
  4. Margin: Espacio externo (separa la caja de otras cajas).

Evaluación 2: Cajas

5. Display Property

Define cómo se comporta la caja en el flujo:

  • block: Ocupa todo el ancho disponible y fuerza un salto de línea (ej. `<div>`, `<p>`).
  • inline: Solo ocupa lo necesario y permite elementos a los lados (ej. `<span>`, `<a>`). No acepta width/height.
  • inline-block: Híbrido. Permite lados y acepta dimensiones.

6. Flexbox Básico

La herramienta definitiva para alinear cosas en una dimensión (fila o columna). 

.padre {
    display: flex; /* Activa poderes mágicos */
    flex-direction: row; /* Fila (por defecto) */
}
Automáticamente, los hijos se ponen uno al lado del otro.

7. Alineación con Flexbox

Cómo centrar un div en 2026: 

.padre {
    display: flex;
    justify-content: center; /* Eje Principal (Horizontal) */
    align-items: center;     /* Eje Cruzado (Vertical) */
    height: 100vh;
}
¡Ya no necesitas trucos raros con márgenes negativos!

8. CSS Grid

Para layouts bidimensionales (filas Y columnas a la vez). 

.galeria {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 columnas iguales */
    gap: 20px; /* Espacio entre celdas */
}
La unidad fr (fracción) es súper potente para crear diseños flexibles.

9. Diseño Responsivo

Tu web debe verse bien en un reloj, un celular y una TV 4K. 

/* Estilos base (Móvil) */
h1 { font-size: 20px; }

/* Si la pantalla es mayor a 768px (Tablet/PC) */
@media (min-width: 768px) {
    h1 { font-size: 40px; }
}
La estrategia Mobile First sugiere diseñar primero para móvil (base) y luego añadir complejidad para pantallas grandes.

10. Posicionamiento

Saca elementos del flujo normal:

  • relative: Se mueve respecto a su posición original, pero deja el hueco.
  • absolute: Se mueve respecto al padre posicionado más cercano. Flota sobre todo.
  • fixed: Se queda pegado a la pantalla al hacer scroll (como un navbar).

11. Pseudo-clases

Estilan un elemento según su estado. 

button:hover {
    background-color: blue; /* Al pasar el mouse */
}

input:focus {
    border-color: green; /* Al escribir dentro */
}
Dan vida y feedback a la interfaz.
