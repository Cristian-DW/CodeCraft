Volver al Curso

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) controla cómo se ven los elementos HTML. Mientras HTML es el esqueleto, CSS es la piel y la ropa. Sintaxis: `selector { propiedad: valor; }`.

2. Colores

Hay varias formas de definir colores:
- Nombre: `red`
- Hexadecimal: `#FF0000`
- RGB: `rgb(255, 0, 0)`
- HSL: `hsl(0, 100%, 50%)`

Evaluación 1: Sintaxis

¿Cuál es la sintaxis correcta para cambiar el color del texto?

p { text-color: red; }
p { color: red; }
p { font-color: red; }

3. Tipografía

Controla las fuentes. `font-family` cambia la letra (Arial, Roboto). `font-size` el tamaño (px, rem). `font-weight` el grosor (bold, 400).

4. Modelo de Caja (Box Model)

Todo en CSS es una caja rectangular.

Evaluación 2: Cajas

¿Qué propiedad controla el espacio INTERNO de la caja?

Margin
Padding
Border

5. Display Property

Determina cómo se comporta la caja.
`block`: Ocupa todo el ancho (ej. div, p).
`inline`: Solo ocupa su contenido (ej. span, a).
`none`: Desaparece.

6. Flexbox Básico

Flexbox es un sistema unidimensional para distribuir espacio. `display: flex` en el padre alinea los hijos en una fila por defecto.

7. Alineación con Flexbox

`justify-content`: Alinea en el eje principal (horizontal).
`align-items`: Alinea en el eje cruzado (vertical).
Es la forma moderna de centrar cosas.

Evaluación 3: Layouts

Para centrar algo perfectamente en el medio con Flexbox usamos:

justify-content: center; align-items: center;
text-align: center; vertical-align: middle;
margin: auto;

8. CSS Grid

Grid es bidimensional (filas y columnas). Es más potente que Flexbox para layouts generales de página. `display: grid; grid-template-columns: 1fr 1fr;`.

9. Diseño Responsivo

Las Media Queries (`@media`) permiten aplicar estilos solo si la pantalla tiene cierto tamaño. Clave para que se vea bien en celulares.

10. Posicionamiento

`relative`: Respecto a sí mismo.
`absolute`: Respecto al padre posicionado más cercano.
`fixed`: Respecto a la ventana del navegador.

11. Pseudo-clases

Estados especiales. `:hover` (cuando pasas el mouse), `:focus` (cuando haces click en un input), `:nth-child` (para seleccionar hijos específicos).

Evaluación Final

¿CSS Grid es mejor para layouts de una sola dimensión?

Verdadero, reemplaza a Flexbox siempre.
Falso, Grid es 2D (filas/col), Flexbox es 1D.
