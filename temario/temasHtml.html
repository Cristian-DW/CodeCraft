Lección 2: Etiquetas, Elementos y Atributos

Etiquetas: Palabras clave entre < > que indican cómo mostrar contenido. Ej: <h1>, <p>.

Elementos: Combinación de etiqueta de apertura, contenido y cierre. Ej: <p>Texto</p>.

Atributos: Información adicional en la etiqueta de apertura. Ej: <a href="...">.

Tema 1: Texto Básico

Lección 3: Estructura de Texto

Encabezados: Jerarquía del contenido (<h1> a <h6>).
Párrafos: Bloques de texto (<p>).
Enlaces: Hipervínculos (<a href="...">).

Tema 4: Accesibilidad

Lección 2: Web para Todos

Hacer la web accesible para personas con discapacidades.

- Usar textos alternativos (alt) en imágenes.
- Usar jerarquía de encabezados correcta.
- Asegurar contraste de colores y navegación por teclado.
