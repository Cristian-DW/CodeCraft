Volver al Curso

1. ¿Qué es HTML?

HTML (HyperText Markup Language) no es un lenguaje de programación, es un lenguaje de marcado. Define la estructura de tu web: qué es un título, qué es un párrafo y qué es una imagen.

2. Etiquetas y Atributos

HTML usa etiquetas rodeadas por corchetes angulares, como `<p>Hola</p>`. Los atributos proveen información extra, como el destino de un enlace.

3. Títulos y Párrafos

Los encabezados van de `h1` (el más importante) a `h6` (el menos importante). Solo debe haber un `h1` por página. Los párrafos usan la etiqueta `p`.

4. Listas

`ul` para listas desordenadas (puntos) y `ol` para ordenadas (números). Cada elemento es un `li`.

5. Enlaces (Links)

La etiqueta más importante de la web: `a`.
`<a href="google.com">Ir a Google</a>` transforma el texto en un hipervínculo.

6. Imágenes

`<img src="foto.jpg" alt="Descripción">`.
Nota que la etiqueta `img` no tiene cierre. El atributo `alt` es crucial para la accesibilidad.

7. HTML Semántico

En lugar de usar `div` para todo, usa etiquetas que signifiquen algo: `header`, `nav`, `main`, `section`, `article`, `footer`. Ayuda a Google a entender tu sitio.

8. Formularios

`<form>` envuelve los controles. `<input type="text">` crea un campo de texto. Existen muchos tipos: `password`, `email`, `date`, `checkbox`.

9. Tablas

Úsalas solo para datos tabulares (como Excel), no para diseño. Estructura: `table` > `tr` (fila) > `th` (encabezado) / `td` (dato).

10. Meta Tags

Van en el `head`. Controlan cosas invisibles pero vitales: el set de caracteres (UTF-8), la descripción para Google y la vista en móviles (viewport).

11. Buenas Prácticas

- Indenta tu código siempre.
- Usa minúsculas para etiquetas.
- Cierra siempre las etiquetas que lo requieran.
- Comenta tu código con `<!-- comentario -->`
