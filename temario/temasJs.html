Volver al Curso

1. ¿Qué es JavaScript?

Es el lenguaje que da vida a la web. Mientras HTML/CSS son estáticos, JS permite interactividad, animaciones y lógica compleja. Se ejecuta en el navegador.

2. Variables Modernas

Olvida `var`. Usa:
`let`: Para valores que cambian (ej. puntuación).
`const`: Para valores fijos (ej. PI, configuraciones).

Evaluación 1: Conceptos Básicos

¿Cuál es la mejor forma de declarar una variable que NO cambiará?

let x = 10;
var x = 10;
const x = 10;

3. Tipos de Datos

Primitivos: String, Number, Boolean, Null, Undefined, Symbol.
Complejos: Object, Array.

4. Operadores

Aritméticos (+, -, *, /).
Comparación (==, ===, >, <).
Lógicos (&&, ||, !).
*Tip:* Usa siempre `===` para comparaciones estrictas.

5. Condicionales

`if (condición) { ... } else { ... }`.
Permite tomar decisiones en el código.

Evaluación 2: Lógica

¿Qué imprime: `if (5 > 10) { console.log("A") } else { console.log("B") }`?

A
B
Nada

6. Bucles (Loops)

Repiten código.
`for`: Cuando sabes cuántas veces repetir.
`while`: Cuando no sabes cuándo parar.

7. Funciones Flecha

Sintaxis moderna: `const sumar = (a, b) => a + b;`

8. El DOM

Document Object Model. Es la representación de tu HTML en memoria.
`document.querySelector('h1')` selecciona el primer título.

9. Eventos

Cualquier cosa que sucede en la web.
`boton.addEventListener('click', funcion)` hace que algo pase al dar click.

Evaluación 3: DOM

¿Qué método usamos para escuchar eventos?

addEventListener
listenEvent
onClick

10. Arrays y Objetos

Arrays: Listas ordenadas `[1, 2, 3]`.
Objetos: Clave-valor `{ nombre: "Juan", edad: 30 }`.

11. Fetch API

Permite hacer peticiones HTTP (AJAX) modernas.
`fetch('https://api.com/datos').then(res => res.json())`

Evaluación Final

¿JavaScript se ejecuta en el servidor o en el cliente?

Solo en el Servidor (con Node.js)
Solo en el Cliente
En ambos (Navegador y Node.js)
