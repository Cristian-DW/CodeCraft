Tema 1: ¿Qué es JSX?

JSX es una extensión de sintaxis para JavaScript que permite escribir HTML dentro de JavaScript. Se parece a una plantilla, pero tiene todo el poder de JS.

Evaluación Rápida

¿JSX es validado directamente por el navegador?

Sí, nativamente.
No, necesita transpilarse (ej. Babel).

Tema 2: Componentes y Props

Los componentes son como funciones de JavaScript que devuelven elementos de React. Aceptan entradas llamadas "props". 

function Bienvenida(props) {
  return <h1>Hola, {props.nombre}</h1>;
}
