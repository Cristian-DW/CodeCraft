Leccion 2: ¿Qué son los lenguajes de programación

Los lenguajes de programación son conjuntos de reglas y símbolos utilizados para escribir instrucciones que una computadora puede entender y ejecutar. Son herramientas fundamentales en el desarrollo de software y permiten a los programadores comunicarse con las máquinas de manera eficiente y precisa. Cada lenguaje de programación tiene su propia sintaxis y semántica, lo que significa que tienen reglas específicas para escribir y organizar el código. Algunos lenguajes son más verbosos y se asemejan más al lenguaje humano, mientras que otros son más concisos y se centran en la eficiencia. Los lenguajes de programación se utilizan para una amplia letiedad de propósitos, desde el desarrollo de aplicaciones de escritorio y móviles, hasta la creación de sitios web, el análisis de datos, la inteligencia artificial y mucho más. Cada lenguaje tiene sus fortalezas y debilidades, y es importante elegir el lenguaje adecuado para cada proyecto, teniendo en cuenta sus requisitos y características.

Tema 2: Estructuras de control

Leccion 1: Condicionales: if, else, else if

Condicionales: if, else if, else. Las estructuras condicionales nos permiten tomar decisiones en función de ciertas condiciones.
Los condicionales más comunes son:

● if: Permite ejecutar un bloque de código si una condición es verdadera.
● else if: Permite verificar una segunda condición si la condición anterior no se cumple.
● else: Se ejecuta cuando ninguna de las condiciones anteriores es verdadera

Tema 2: Estructuras de control

Leccion 2: Bucles: for, while.

Bucle for: El bucle for es una estructura de control que permite repetir un bloque de código un número específico de veces. Se compone de tres partes: la inicialización, la condición de continuación y la expresión de incremento o decremento. El bucle for se utiliza cuando se conoce la cantidad exacta de iteraciones necesarias.

Bucle while: El bucle while es otra estructura de control que repite un bloque de código mientras una condición dada sea verdadera. A diferencia del bucle for, el bucle while se utiliza cuando no se conoce la cantidad exacta de iteraciones y se repite hasta que la condición se vuelva falsa.

Leccion 3: Uso de funciones.

Las funciones son bloques de código que se pueden llamar o invocar en cualquier parte de un programa. Su uso principal es organizar y reutilizar código. Algunos puntos importantes sobre las funciones son:

● Definición de función: Se declara utilizando la palabra clave "function", seguida del nombre de la función y los parámetros que acepta.
● Parámetros y argumentos: Los parámetros son variables utilizadas por la función, mientras que los argumentos son los valores reales pasados a la función al invocarla.
● Valor de retorno: Una función puede devolver un valor mediante la palabra clave "return". Este valor puede ser utilizado en el programa principal.

Leccion 2: Encapsulamiento, herencia y polimorfismo.

Estos son tres conceptos clave en la programación orientada a objetos:

● Encapsulamiento: Es el proceso de ocultar los detalles internos de un objeto y proporcionar una interfaz para interactuar con él. Esto se logra mediante la definición de atributos y métodos públicos y privados.
● Herencia: Permite crear nuevas clases basadas en clases existentes. La herencia permite que una clase herede atributos y métodos de otra, promoviendo la reutilización del código y la creación de jerarquías de clases.
● Polimorfismo: Es la capacidad de un objeto de comportarse de diferentes formas según el contexto. Permite que diferentes objetos respondan de manera distinta a una misma llamada a un método, facilitando la flexibilidad y extensibilidad del código.
